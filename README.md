# MapleStory Typing Game
※ This README is written in **Japanese first**, with **English explanations below each section**.

---

## 概要 / Overview

### 日本語
MapleStory の用語集を用いた、ブラウザ上で遊べるタイピングゲームです。  
画面に表示されたハングルを見てローマ字で入力し、正解すると対応する日本語表記と画像が表示されます。

言語学習要素とゲーム性を組み合わせた、無料の Web タイピングゲームとして開発しています。

### English
This is a free, browser-based typing game using MapleStory terminology.  
Players see Hangul words, type answers in romaji, and upon correct input, the corresponding Japanese text and an image are displayed.

The project combines language learning with simple typing gameplay.

---

## 主な機能 / Features

### 日本語
- ハングル → ローマ字入力によるタイピング
- 正解時の演出
  - 日本語テキストの表示
  - 関連画像の表示
  - 簡単な正解アニメーション
- MapleStory 関連用語を使用（非公式・ファンメイド）
- HTML / CSS / JavaScript のみで動作
- UTF-8 対応（ハングル・日本語・ローマ字）

### English
- Hangul to romaji typing gameplay
- Correct-answer effects:
  - Japanese text display
  - Related image display
  - Simple animation
- Uses MapleStory-related terminology (fan-made)
- Lightweight implementation using only HTML / CSS / JavaScript
- UTF-8 multilingual support (Hangul / Japanese / Romaji)

## 遊び方 / How to Play

### 日本語
1. 画面にハングルの単語が表示されます。
2. ローマ字で正しい読みを入力します。
3. 正解すると、日本語訳と画像が表示されます。
4. 次の問題へ進みます。

### English
1. A Hangul word is shown on the screen.
2. Type the correct answer using romaji.
3. When the input is correct, the Japanese translation and an image are displayed.
4. Proceed to the next question.
